## Install

Keep everything clean in one folder with no outside dependencies except npm.

```bash
# install
cd pi
npm install @mariozechner/pi-coding-agent

# update
npm install @mariozechner/pi-coding-agent

node /app/pi/node_modules/@mariozechner/pi-coding-agent/dist/cli.js --config-dir ./.pi
```
